## ğŸš€ Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/JenxxAI/AHFJC-Church-Website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd AHFJC-Church-Website
   ```

3. Open `index.html` in your web browser, or use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

4. Visit `http://localhost:8000` in your browser

## ğŸ“ Customization

### Adding Images

Place your images in the `assets/images/` folder and update the HTML to reference them:

```html
<img src="assets/images/your-image.jpg" alt="Description">
```

### Modifying Styles

The CSS is organized into modular files for easier maintenance:
- **base.css** - Global styles and reset
- **Section-specific files** - Each section has its own CSS file
- **responsive.css** - All media queries for mobile/tablet responsiveness

Edit the specific CSS file for the section you want to change. See [assets/css/README.md](assets/css/README.md) for details.

### Adding Functionality

Add custom JavaScript in `assets/js/main.js` for interactive features.
