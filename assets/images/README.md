# Images Folder

This folder contains all image assets for the AHFJCPI website.

## Recommended Image Organization:

- `logo/` - Church logos and branding
- `churches/` - Photos of church buildings
- `events/` - Event photos
- `ministry/` - Ministry activity photos
- `gallery/` - General photo gallery
- `leadership/` - Leadership photos
- `icons/` - Icon files

## Image Guidelines:

- Use optimized images (compressed for web)
- Recommended formats: JPG for photos, PNG for graphics with transparency, SVG for logos
- Use descriptive filenames (e.g., `sagay-church-exterior.jpg`)
- Maximum recommended size: 2MB per image
